## Project Overview

A simple React frontend project demonstrating responsive design, interactive components, and clean code structure.

## Features

* Responsive layout
* Interactive components (buttons, forms, checkboxes)
* Modular and maintainable code
* Modern React practices using hooks and functional components

## Installation

1. Clone the repository:

```bash
git clone https://github.com/amirahaha28/tdcx-frontend.git
```

2. Navigate into the project folder:

```bash
cd tdcx-frontend
```

3. Install dependencies:

```bash
yarn install
```

4. Start the development server:

```bash
yarn dev
```

5. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Technologies Used

* React
* JavaScript (ES6+)
* HTML5 & CSS3
* Vite (bundler & development server)
* Material UI
